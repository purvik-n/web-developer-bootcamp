Who Am I?

Hi! I'm Purvik N Gowda, an aspiring web developer currently working through The Web Developer Bootcamp. This repository documents my learning journey — every commit represents a concept mastered, a bug fixed, or a new skill picked up.

What is This Repository?

This is my HTML practice repository. It serves as a living record of everything I am studying, from basic document structure all the way to semantic markup, accessibility best practices, and multi-page site navigation.

HTML Concepts Covered So Far

  1. Document Structure

    Understanding <!DOCTYPE html>, <html>, <head>, and <body> tags.

  2. Headings & Paragraphs

    Using <h1>–<h6> for hierarchy and <p> for body copy.

  3. Text Formatting

    Bold (<b> / <strong>), italics (<i> / <em>), and inline code (<code>).

  4. Links & Navigation

    Absolute vs. relative URLs using <a href>, opening links in new tabs with target="_blank", and internal anchor links.

  5. Images

    Embedding images with <img>, using src, alt, and width attributes for accessibility.

  6. Lists

    Ordered (<ol>), unordered (<ul>), and description lists (<dl>, <dt>, <dd>), including nested lists.

  7. Tables

    Building data tables with <table>, <thead>, <tbody>, <tr>, <th>, and <td>.

  8. Semantic & Structural Elements

    Using <header>, <nav>, <main>, <section>, <article>, and <footer> for meaningful page structure.

  • Introduction to CSS styling
  • Responsive design fundamentals
